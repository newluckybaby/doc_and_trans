选择合适的终端环境

在Linux中，相比GUI工具，CLI往往更得心应手一些，所以选择一个顺手的Terminal环境是尤为重要的。ubuntu下常见的终端环境有gnome-termial,xterm以及rxvt，各有特色。

gnome-terminal

gnome-terminal是较为常见的，和默认的ubuntu桌面环境集成，不需要安装，字体渲染最好。

xterm

xterm默认不支持cjk字符，通常情况下安装xterm会安装名为uxterm的shell文件。直接执行uxterm即可。xterm参数较多，可定制性也要强一些。附xterm配置文件。
